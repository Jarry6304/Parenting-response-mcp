# cores(v3.0)— 6 回應核心 + 2 探詢核心 + safety 安全約束集

> 學派分兩功能群:**回應核心(6)** 產 posture TAG,③ 回傳給 host 耦合;**探詢核心(2:maslow, satir)** 產診斷探點,① 約束探詢回傳,引導 S1 理解情境,**不進回應耦合**。本檔為單一事實來源。

## 兩種格式

```text

# 探詢核心
<school>: { 探詢, 探點, 示範問, 紅線 }
```

## 回應核心（6,進 ③ 耦合）

### pd（正向教養 / Positive Discipline）

```text
pd:
  理念: 和善與堅定並行;不當行為是「喪氣的孩子」在找歸屬與價值,錯誤是學習機會。
  套用: 先連結情感再矯正行為(connection before correction);用有限選擇、邏輯/自然後果(非處罰)、共同解題,把主導權與責任交還孩子。
  示範: 「我看到你很難過。我們一起想想,下次可以怎麼做?」
  紅線: 不羞辱、不處罰、不用獎懲操控、不替孩子承擔後果、不在情緒高點說教。
```

### adler（阿德勒 / 個體心理學）

```text
adler:
  理念: 人追求歸屬感與意義;不當行為是喪氣(discouragement)的表現,鼓勵是核心解方。
  套用: 用鼓勵(聚焦努力與貢獻,非讚美結果)建立歸屬與價值;邀請孩子貢獻、賦予責任。
  示範: 「你自己想辦法把鞋帶綁好了,你很努力。」(鼓勵努力,非「你好棒」)
  紅線: 不打擊、不過度讚美結果、不溺愛代勞、不製造自卑。
```

### dreikurs（Dreikurs / 行為四目的 + 邏輯後果）

```text
dreikurs:
  理念: 不當行為服務四個錯誤目的之一:尋求關注、權力鬥爭、報復、自暴自棄。對症不對行為。
  套用: 從家長的情緒反推目的(煩→關注 / 怒→權力 / 傷→報復 / 無望→自暴自棄),以邏輯後果(相關、尊重、合理 3R)回應而非處罰。
  示範: 「你把牛奶打翻了,這裡有抹布,我們一起擦乾淨。」(邏輯後果)
  紅線: 不捲入權力鬥爭、不報復式懲罰、不羞辱;後果不可帶懲罰或羞辱性。
```

### gottman（Gottman / 情緒教練）

```text
gottman:
  理念: 情緒沒有對錯,行為才有界線;負面情緒是親近與教導的契機。
  套用: 五步:覺察情緒→視為契機→同理反映→協助命名情緒→情緒被接住後才設限與解題。
  示範: 「你現在很生氣,因為積木倒了對不對?生氣是可以的,我們先深呼吸。」
  紅線: 不否定/貶低情緒、不在情緒未被接住前急著矯正講道理、不嘲笑情緒。
```

### rogers（Rogers / 個人中心）

```text
rogers:
  理念: 人有自我實現傾向;在真誠、無條件積極關注、同理的關係中,孩子能自我調節與成長。
  套用: 提供三條件;先反映、確認、接住,不急著矯正或給答案;必要時退場降溫。
  示範: 「聽起來今天在學校讓你覺得很委屈。我在這裡聽你說。」
  紅線: 不評價、不條件式關愛(乖才愛)、不替孩子下結論、不在未被理解前急著建議。
```

### nvc（非暴力溝通 / Nonviolent Communication）

```text
nvc:
  理念: 一切行為都在滿足某種普世需求;衝突是策略之爭,非需求之爭。
  套用: 四步:觀察(不評價)→感受→需求→請求(具體、可行、正向);先同理孩子,再表達自己。
  示範: 「玩具散在地上(觀察),我有點擔心(感受),我需要安全走道(需求),你願意現在收到箱子嗎?(請求)」
  紅線: 不評價貼標籤、不命令威脅、不把請求變要求(拒絕即懲罰)、不歸咎。
```

## 探詢核心（2,屬 ① 約束探詢,不進耦合）

### maslow（需求層次）

```text
maslow:
  探詢: 未滿足的缺損需求(生理 / 安全 / 愛與歸屬 / 尊重)是否在驅動此行為。
  探點: 餓?睏?累?沒安全感?沒被歸屬或尊重?(兒少聚焦缺損四層,不談自我實現)
  示範問: 「你是不是太累了 / 肚子餓了?」(先排除缺損,再談行為)
  紅線: 不在基本需求未滿足時硬要求高層行為(餓著睏著還逼守規矩)、不忽略生理訊號。
```

### satir（薩提爾 / 冰山理論）

```text
satir:
  探詢: 冰山下層——這個行為下,孩子真正的感受 / 期待 / 渴望。
  探點: 表面行為 vs 水面下渴望(被看見?被當大人?自主?連結?)
  示範問: 「你這樣做,是不是其實希望___?」(探渴望,不只處理表面)
  紅線: 不指責、不超理智說教、不討好妥協原則、不只處理表面行為。
```

## 覆盤視角(v3.0 B 件;retro mode ③ 回傳,僅 6 回應核心)

> retro = 事後覆盤:家長交代「當時實際怎麼處理」(`parent_action`),③ 回六校
> **覆盤鏡頭**——各校用自身理論回看那次處理,給「回看點 → 改寫操作 → 示範」。
> 探詢核心(maslow/satir)不設覆盤塊:其角色是探詢孩子內心,屬 ① 約束探詢,
> 不作家長行為的覆盤視角。鍵:`視角 / 操作 / 示範`,6 塊缺一 fail-fast。

### pd.覆盤

```text
pd.覆盤:
  視角: 回看當時是否先連結情感才矯正行為?處理是「和善且堅定」並行,還是偏向一端(只嚴厲/只退讓)?
  操作: 找出一個「下次可以更早連結」的時點;把當時的處罰或讓步,改寫成有限選擇或自然後果。
  示範: 「當時我直接命令他。下次我可以先蹲下來說『你看起來很失望』,再給兩個選擇。」
```

### adler.覆盤

```text
adler.覆盤:
  視角: 回看孩子當時的行為背後是哪裡喪氣了?我的回應是鼓勵(看見努力與貢獻),還是打擊或只評價結果?
  操作: 找出孩子當時值得鼓勵的一個努力點;把評價結果的話,改寫成描述努力的話。
  示範: 「我只看到他打翻牛奶,沒看到他想自己倒的嘗試。下次先說『你想自己來,這個想法很好』。」
```

### dreikurs.覆盤

```text
dreikurs.覆盤:
  視角: 用我當時的情緒反推孩子的錯誤目標(煩→關注/怒→權力/傷→報復/無望→自暴自棄);我的回應是否正中目標(怒而硬壓=捲入權力鬥爭)?
  操作: 確認目標後,設計一個「不餵目標」的回應:關注型給預約關注、權力型給選擇權、報復型先修復關係、自暴自棄型把任務拆小。
  示範: 「我當時很怒,代表他在爭權力;我硬壓他,正好捲入鬥爭。下次給選擇:現在收,或十分鐘後收。」
```

### gottman.覆盤

```text
gottman.覆盤:
  視角: 回看情緒教練五步斷在哪一步:有覺察情緒嗎?有當作親近契機嗎?有同理、有幫他命名嗎?還是情緒沒被接住前就急著設限說教?
  操作: 標出斷掉的那一步,補寫那一步「當時可以說的一句話」;界線可以保留,順序放在情緒被接住之後。
  示範: 「我跳過了命名直接講道理。下次先說『你很生氣,因為你還想玩』,等他點頭再談規則。」
```

### rogers.覆盤

```text
rogers.覆盤:
  視角: 回看孩子有沒有「先被理解」的經驗:我是反映他的感受,還是急著評價、給答案、下結論?我的關注是無條件的,還是「乖才有」?
  操作: 把當時最早說出口的那句話改寫成反映句(「聽起來你⋯⋯」);若當時自己已失溫,標記應退場降溫的時點。
  示範: 「我第一句是『這有什麼好哭』。下次改成『聽起來真的很委屈』,先讓他知道我聽到了。」
```

### nvc.覆盤

```text
nvc.覆盤:
  視角: 回看我當時說的話:是觀察還是評價?有說出感受與需求,還是只有命令?我的「請求」是不是其實是要求(不從就罰)?
  操作: 把當時的話拆回四步重寫:具體觀察、我的感受、我的需求、可行的請求;並檢查孩子若拒絕,我接不接得住。
  示範: 「『你又把家裡弄得一團亂』是評價。改寫:『積木散在客廳(觀察),我擔心踩到(感受),我需要走道安全(需求),睡前收回箱子好嗎?(請求)』」
```

## safety 安全約束集(v3.0 G 件;③ safety_mode 組卡素材)

> 結構:**3 風險向底座**(child / parent / third,內容本質不同)× **4 年齡 delta**(只調語域與姿態,僅疊加於 child 向——parent/third 的內容對象非孩子)。風險向由 G0 命中組自然攜帶(`wordlists.G0_SHORTCIRCUIT_BY_VECTOR`),server 不另判斷。7 塊全顯式、缺一 fail-fast;delta 可標「無補充」但塊必須存在。每塊 `source` 錨定已出版準則——改寫幅度限於語域調整,不自創方法。

### safety.base.child（孩子自身風險:自傷/自殺意念、傷人傷物升級）

```text
safety.base.child:
  承接: 先穩住自己的語氣與表情,不驚呼、不斥責;孩子說出口是求助訊號,被接住才會繼續說。
  不否定: 不說「不要亂講」「你想太多」「不准再說這種話」——否定會關上門,讓下次不再開口。
  直接問: 平靜、直接地問「你是不是有想過傷害自己?」;實證共識是直接詢問不會「提醒」或誘發,反而打開對話(守門人「1問」)。
  回應: 聽完不急著給建議或說教;反映你聽到的、肯定他願意說(守門人「2應」)。
  行動: 不讓孩子獨處;移開危險物品;儘速連結專業——學校輔導室、兒童青少年心智科、安心專線 1925、生命線 1995、張老師 1980(守門人「3轉介」);立即危險撥 110/119。
  source: 衛福部自殺防治守門人「1問2應3轉介」;全國自殺防治中心守門人宣導素材。
```

### safety.base.parent（家長端風險:瀕臨失控、傷害衝動、自身耗竭)

```text
safety.base.parent:
  安住自己: 此刻優先序是你的狀態,不是孩子的行為——先停下管教,深呼吸十秒、喝口水;這是安全程序,不是失格。
  物理隔開: 確認孩子在安全處後,暫時離開現場到另一個空間;距離是保護彼此,不是逃避。
  求援: 立即聯繫另一半、家人或可信任的人接手;安心專線 1925(24 小時)可即時通話;若已失手或衝動強烈反覆出現,主動就醫或撥 113 求助——先求助的人是在保護孩子。
  自身傷害意念: 「撐不下去、活不下去」也適用同一套——直說、求援、不獨處;1925/1995 對家長同樣開放。
  source: 衛福部安心專線 1925;113 保護專線;自殺防治守門人原則(對家長自身之投影)。
```

### safety.base.third（第三方加害/疏忽、孩子目睹暴力)

```text
safety.base.third:
  確保安全: 第一優先是讓孩子離開危險現場與接觸源;不需先查證屬實才行動,安全疑慮即足以啟動。
  通報義務: 知悉兒少疑似受虐、疏忽、不當對待或目睹家庭暴力,依兒童及少年福利與權益保障法第 53 條應通報——撥 113 保護專線(24 小時,可匿名諮詢);緊急危險撥 110。
  不質問不重演: 記下時間、地點與你觀察到的原話;不反覆質問細節、不要求孩子重演或對質(避免二度傷害,細節查證交給專業)。
  接住目睹者: 對目睹的孩子明確說「這不是你的錯」「你說出來是對的」;後續可連結各縣市家防中心之目睹兒少服務。
  source: 兒童及少年福利與權益保障法第 53 條;113 保護專線;衛福部目睹家暴兒少服務資源(各地家庭暴力暨性侵害防治中心)。
```

### safety.delta.2-3（幼齡:語域與姿態補充)

```text
safety.delta.2-3:
  補充: 以身體安撫與極短句為主——抱住、拍背、「我在」;此齡死亡概念未成形,勿用抽象詞彙追問「想不想活」,改以具體行為觀察(咬自己、撞頭、長時間退縮)描述並交專業評估。
  source: 守門人分齡溝通原則之語域投影;幼兒發展之死亡概念研究通識(概念理解約自學齡期漸次成形)。
```

### safety.delta.4-6（學前:語域與姿態補充)

```text
safety.delta.4-6:
  補充: 用具體、簡單的詞彙,蹲到孩子視線高度;可借遊戲、畫畫、玩偶讓孩子表達;「想消失」類語言先當求助訊號承接與好奇(「你說想消失,可以多告訴我一點嗎?」),不自行論斷輕重,交專業評估。
  source: 守門人分齡溝通原則之語域投影;遊戲為學前兒主要表達媒介之發展共識。
```

### safety.delta.7-11（學齡:語域與姿態補充)

```text
safety.delta.7-11:
  補充: 不說教、不在手足或同學面前處理;明確說「這不是你的錯」「跟我說不會被罵」;維持日常作息與陪伴給安全感;可引導用寫的或畫的表達說不出口的部分。
  source: 守門人分齡溝通原則之語域投影;學齡期同儕自尊敏感之發展共識。
```

### safety.delta.12+（青少年:語域與姿態補充)

```text
safety.delta.12+:
  補充: 尊重自主與隱私,以平等姿態直接問(「你最近是不是有過不想活的念頭?」)——12+ 直接問是實證支持的做法;不沒收手機作為此事的回應(切斷求助與同儕支持管道);就醫與求助給選項、邀請而非押送;承諾「陪你一起」而非「我幫你決定」。
  source: 衛福部自殺防治守門人「1問2應3轉介」(青少年直接詢問之實證共識);校園自殺防治守門人概念(教育部校園學生自我傷害三級預防)。
```
