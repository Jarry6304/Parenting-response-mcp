# cores（v3.0）— 6 回應核心 + 2 探詢核心

> 學派分兩功能群:**回應核心(6)** 產 posture TAG,③ 回傳給 host 耦合;**探詢核心(2:maslow, satir)** 產診斷探點,① 約束探詢回傳,引導 S1 理解情境,**不進回應耦合**。本檔為單一事實來源。

## 兩種格式

```text
# 回應核心

# 探詢核心
<school>: { 探詢, 探點, 示範問, 紅線 }
```

## 回應核心（6,進 ③ 耦合）

### pd（正向教養 / Positive Discipline）

```text
pd:
  理念: 和善與堅定並行;不當行為是「喪氣的孩子」在找歸屬與價值,錯誤是學習機會。
  套用: 先連結情感再矯正行為(connection before correction);用有限選擇、邏輯/自然後果(非處罰)、共同解題,把主導權與責任交還孩子。
  示範: 「我看到你很難過。我們一起想想,下次可以怎麼做?」
  紅線: 不羞辱、不處罰、不用獎懲操控、不替孩子承擔後果、不在情緒高點說教。
```

### adler（阿德勒 / 個體心理學）

```text
adler:
  理念: 人追求歸屬感與意義;不當行為是喪氣(discouragement)的表現,鼓勵是核心解方。
  套用: 用鼓勵(聚焦努力與貢獻,非讚美結果)建立歸屬與價值;邀請孩子貢獻、賦予責任。
  示範: 「你自己想辦法把鞋帶綁好了,你很努力。」(鼓勵努力,非「你好棒」)
  紅線: 不打擊、不過度讚美結果、不溺愛代勞、不製造自卑。
```

### dreikurs（Dreikurs / 行為四目的 + 邏輯後果）

```text
dreikurs:
  理念: 不當行為服務四個錯誤目的之一:尋求關注、權力鬥爭、報復、自暴自棄。對症不對行為。
  套用: 從家長的情緒反推目的(煩→關注 / 怒→權力 / 傷→報復 / 無望→自暴自棄),以邏輯後果(相關、尊重、合理 3R)回應而非處罰。
  示範: 「你把牛奶打翻了,這裡有抹布,我們一起擦乾淨。」(邏輯後果)
  紅線: 不捲入權力鬥爭、不報復式懲罰、不羞辱;後果不可帶懲罰或羞辱性。
```

### gottman（Gottman / 情緒教練）

```text
gottman:
  理念: 情緒沒有對錯,行為才有界線;負面情緒是親近與教導的契機。
  套用: 五步:覺察情緒→視為契機→同理反映→協助命名情緒→情緒被接住後才設限與解題。
  示範: 「你現在很生氣,因為積木倒了對不對?生氣是可以的,我們先深呼吸。」
  紅線: 不否定/貶低情緒、不在情緒未被接住前急著矯正講道理、不嘲笑情緒。
```

### rogers（Rogers / 個人中心）

```text
rogers:
  理念: 人有自我實現傾向;在真誠、無條件積極關注、同理的關係中,孩子能自我調節與成長。
  套用: 提供三條件;先反映、確認、接住,不急著矯正或給答案;必要時退場降溫。
  示範: 「聽起來今天在學校讓你覺得很委屈。我在這裡聽你說。」
  紅線: 不評價、不條件式關愛(乖才愛)、不替孩子下結論、不在未被理解前急著建議。
```

### nvc（非暴力溝通 / Nonviolent Communication）

```text
nvc:
  理念: 一切行為都在滿足某種普世需求;衝突是策略之爭,非需求之爭。
  套用: 四步:觀察(不評價)→感受→需求→請求(具體、可行、正向);先同理孩子,再表達自己。
  示範: 「玩具散在地上(觀察),我有點擔心(感受),我需要安全走道(需求),你願意現在收到箱子嗎?(請求)」
  紅線: 不評價貼標籤、不命令威脅、不把請求變要求(拒絕即懲罰)、不歸咎。
```

## 探詢核心（2,屬 ① 約束探詢,不進耦合）

### maslow（需求層次）

```text
maslow:
  探詢: 未滿足的缺損需求(生理 / 安全 / 愛與歸屬 / 尊重)是否在驅動此行為。
  探點: 餓?睏?累?沒安全感?沒被歸屬或尊重?(兒少聚焦缺損四層,不談自我實現)
  示範問: 「你是不是太累了 / 肚子餓了?」(先排除缺損,再談行為)
  紅線: 不在基本需求未滿足時硬要求高層行為(餓著睏著還逼守規矩)、不忽略生理訊號。
```

### satir（薩提爾 / 冰山理論）

```text
satir:
  探詢: 冰山下層——這個行為下,孩子真正的感受 / 期待 / 渴望。
  探點: 表面行為 vs 水面下渴望(被看見?被當大人?自主?連結?)
  示範問: 「你這樣做,是不是其實希望___?」(探渴望,不只處理表面)
  紅線: 不指責、不超理智說教、不討好妥協原則、不只處理表面行為。
```
