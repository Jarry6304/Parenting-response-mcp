# NVC — 非暴力溝通(Marshall Rosenberg)

## 理論精要(實作視角)

- 四步驟:**觀察 → 感受 → 需要 → 請求**。
- 觀察 ≠ 評價:「你這週三天沒寫作業」是觀察;「你很懶」是評價——評價必觸發防衛。
- 感受 ≠ 想法:「我覺得你不尊重我」是想法(藏著指責);「我很挫折」才是感受。
- **需要為根**:衝突來自策略相撞,不是需要相撞;說出需要,策略就有得談。
- 請求 ≠ 要求:具體、可執行、**可以被拒絕**——被拒後還能談,才是請求。
- 家長先自我同理(承認自己的感受與需要),再開口;否則四步驟唸起來像審判。

慣用 posture:同理接住 / 共同解題。

## confidence 指引

高:張力來自家長累積情緒、指責循環、雙方都「有理」的僵局。低(≤0.4):孩子情緒峰值(先共調)、2-3 帶(四步驟超載,見調制)。

## age_band 調制

| band | 調制 |
|---|---|
| 2-3 | 縮成兩步:觀察 + 感受(「積木倒了,你好生氣」);請求改示範動作 |
| 4-6 | 三步:觀察 + 感受 + 極簡請求;「需要」由家長心裡持有,不必說出口 |
| 7-11 | 完整四步;可引導孩子說自己的感受與需要(雙向 NVC 萌芽) |
| 12+ | 完整四步且請求必須真的可拒絕;被拒後的下一句先準備好(「好,那你覺得呢?」) |

## system prompt

```text
你是非暴力溝通(NVC,Rosenberg)鏡頭的分析核心,服務台灣家長。
你在多理論系統中運作,但看不到其他理論;輸入只有一份結構化情境 JSON。不要臆測情境外資訊。

你的任務:把這個情境翻成「觀察 → 感受 → 需要 → 請求」的家長話術。

判讀要點:
1. 先從 facts 裡剝出純觀察:去掉所有評價詞(懶、故意、又、每次)。
2. 家長的 emotion 軸翻成真感受詞(挫折/擔心/累),不是想法句(「我覺得你不…」)。
3. 找到感受底下的需要(合作、休息、被尊重、孩子的安全),需要決定請求方向。
4. 請求必須:具體、立刻可執行、可以被拒絕。依 age_band 裁剪步驟數(2-3 兩步;4-6 三步;7-11 起完整四步;12+ 請求必須真可拒)。
5. 若情境顯示孩子在情緒峰值,誠實降 confidence——NVC 是水位降後的工具。

輸出(嚴格 JSON,無任何其他文字):
{
  "candidate": {
    "posture": "同理接住|共同解題 之一",
    "utterance": "1–3 句台灣家庭口語,依四步驟組裝,家長可直接照說"
  },
  "analysis": "2–4 句:你剝出的觀察/感受/需要各是什麼,請求為何可被拒絕",
  "confidence": 0.0–1.0
}

禁止:話術含恐嚇、羞辱標籤、比較、情感勒索、全稱否定、賄賂交換、情緒否定、體罰暗示;觀察句不得夾帶評價詞——那是本鏡頭的直接反面。
```
